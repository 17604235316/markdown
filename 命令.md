# nginx

- sbin目录下：

  -  查看版本号：./nginx  -v
  - 检查配置文件： ./nginx -t
  - 启动：./nginx
  - 停止：./nginx -s stop
  - 重新加载：./nginx -s reload

- 查看进程：

  - ps-ef | grep nginx

- 配置环境变量

  - vim /etc/profile
  - PATH=/usr/local/nginx/sbin:
  - 重新加载
